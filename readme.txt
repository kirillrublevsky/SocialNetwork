This application is a simulation of real social network like
Vkontakte or Facebook. It allows you to perform next actions:

-write messages;
-add/remove friends;
-add/remove hobbies;
add/remove visited places;

Technologies used in this project:
Java 7, Spring, Hibernate, Spring MVC, H2 in-memory database,
Maven, Git, Html, JavaScript.

Project is not finished yet.
